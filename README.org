*** graalvm

*** lein-graal

The above ~graalvm~ image with [[https://github.com/technomancy/leiningen][Leiningen]] and Make installed.

Use like this:

#+BEGIN_SRC sh
  docker run -ti --rm                                                   \
         --name lein-graal-build                                        \
         -v "$(pwd):/app:ro"                                            \
         -v "$(pwd)/target/default+uberjar:/app/target/default+uberjar" \
         --workdir /app                                                 \
         runejuhl/lein-graal                                            \
         lein native-image
#+END_SRC
